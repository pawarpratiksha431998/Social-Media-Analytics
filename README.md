# Social-Media-Analytics
Project focuses on evaluating digital ad performance across Google and Facebook platforms to identify optimization opportunities and measure marketing effectiveness. campaign data was analyzed to derive actionable insights into engagement, conversion, and ROI performance.
# Tools
Excel | SQL | Power BI | Tableau
# Highlights
- Built dashboards for engagement, audience growth, and campaign performance.
- Wrote SQL queries for KPIs like CTR, conversions, and ROAS.
- Analyzed audience and ad performance to optimize marketing outcomes.
- Delivered data-driven insights to improve campaign strategy and ROI.
